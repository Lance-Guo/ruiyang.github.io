1.新的结构标签和属性

结构标签 用法
<header> 定义页面或区段的头部
<footer> 定义页面或者区段的尾部
<nav> 定义页面或者区段的导航区域
<section> 定义页面的逻辑区域或内容组合
<article> 定义正文或一片完整的内容
<aside> 定义补充或相关内容
<meter> 描述指定范围内的数值
<progress> 用于显示实时进度的控件
自定义的数据属性 允许通过“data-”前缀向任意元素中添加自定义的数据属性
